# Customer Segmentation - RFM Analysis

## Overview
This project uses **RFM (Recency, Frequency, Monetary)** analysis to segment customers from a marketing campaign dataset into groups (VIP, Loyal, Potential, At Risk), helping businesses design targeted marketing strategies.

## Dataset
- **File**: `marketing_campaign.csv`
- Each row represents a customer profile including demographics, purchase behavior, and campaign responses.

## Tech Stack
- Python
- Pandas

## Installation
pip install pandas
jupyter notebook RFM_CUSTOMER_SEGMENT.ipynb
